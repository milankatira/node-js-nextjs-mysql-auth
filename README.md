### to run this project 

#### for api

- install node modules
```bash
yarn
```

- start server 
```bash
yarn dev
```

#### for client

```bash
cd my-app
```


- install node modules
```bash
yarn
```

- start server 
```bash
yarn dev
```

- node version mestion in nvmrc if face any error select node version using

```bash
nvm use
```
